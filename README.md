# Airship – פרסום על העננים 🎈

אתר תדמית מודרני, נקי ומינימליסטי עבור **Airship (איירשיפ)** – חברת פרסום אווירי
ישראלית הפעילה משנת 2006. האתר נבנה מחדש במקום אתר ה-FrontPage הישן, בעיצוב
"שמיימי" אוורירי עם אנימציות עדינות שמשדרות תחושת קלילות וריחוף.

## ✨ מה כולל האתר

- **דף נחיתה אחד (Single Page)** עם גלילה חלקה בין הקטעים וניווט עליון דביק.
- **RTL מלא ועברית**, פונט `Heebo` מ-Google Fonts.
- **אנימציות עדינות**: ריחוף ספינת אוויר, חשיפת תוכן בגלילה (Intersection Observer),
  זום עדין בגלריה, וכפתור וואטסאפ צף עם פעימה.
- **רספונסיבי** (מובייל-פירסט) ונגיש: טקסט חלופי לתמונות, ניגודיות טובה,
  מצבי פוקוס למקלדת, וכיבוד `prefers-reduced-motion`.
- **SEO**: כותרות ותגיות מטא בעברית, תגיות Open Graph, ו-favicon מהלוגו.

## 🗂️ מבנה הקבצים

```
.
├── index.html      # מבנה הדף (סמנטי: header / nav / main / section / footer)
├── styles.css      # עיצוב, צבעים, אנימציות ורספונסיביות
├── script.js       # ניווט דביק, תפריט מובייל, חשיפת גלילה
└── assets/         # תמונות ולוגו
```

## 🚀 הרצה מקומית

זהו אתר סטטי – אין צורך בהתקנות. אפשר פשוט לפתוח את `index.html` בדפדפן,
או להריץ שרת מקומי קטן:

```bash
python3 -m http.server 8000
# ואז לגלוש אל http://localhost:8000
```
